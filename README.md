## Installation

1. Clone the repo:
   ```bash
   git clone https://github.com/TahaAhmed-hup/filmhub.git
   ```
2. Go into project folder:
   ```bash
   cd filmhub
   ```
3. Install dependencies:
   ```bash
   flutter pub get
   ```
4. Run the app:
   ```bash
   flutter run
   ```
